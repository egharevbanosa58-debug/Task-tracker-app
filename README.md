## Getting Started
Run the app and type in any task of your choice.
Click the Add button (+) to add tasks to your dashboard.
You can use the filter buttons (All, Active and Completed) to customize your view.
Use the checkboxes to keep track of completed tasks.
You can use the delete button to detele unwanted tasks.
